Connect Online Registration

The registration form is ready. It only needs Firebase Realtime Database details in firebase-config.js so phones scanning the QR code can save registrations into the shared online list.

What To Do

  1. Create a Firebase project.
  2. Add a Web App in that Firebase project.
  3. Copy the Firebase web app config.
  4. Open firebase-config.js in this GitHub repository.
  5. Replace every PASTE_... value with the values from Firebase.
  6. Enable Firebase Realtime Database.
  7. Use the test rules below while setting it up.

Temporary Test Rules

{
  "rules": {
    "registrations": {
      ".read": true,
      ".write": true
    }
  }
}

After Setup

Open register.html again. The red warning should disappear and players can submit registrations. The main system will automatically import new registrations into the roster, or you can press Sync Registrations in the Players screen.
